# edusafesystem_helene_clara

Descritpion du projet :

Nous avons crée un système de detection de température qui va envoyer une notification sur Tago.io lorsque la température est supérieure à 30°C. Nous l'avons inclus dans une salle de classe en référence à nos études actuelle pour devenir professeur des écoles. 

Matériel utilisé : 
- carte UCA
- arduino pour le code
- carton + légo pour la maquette

perspéctive possible : 
1/ buzzer (ce qu'on avait prévu de base mais qui n'a malheureusement pas fonctionné) qui s'activerai avant l'envoie du message. 
2/ detecteur de fumée à la place d'un détecteur de température
